WEB APPLICATION

หน้า URBAN STREET CANYON(USC) ออกแบบมาเพื่อแสดงผลการศึกษาเกี่ยวกับการจำ
แนกประเภทของหุบเมืองด้วยแบบจำลองการเรียนรู้เชิงลึกในเขตพื้นที่ศึกษา (เทศบาลนครเชียงใหม่) และแสดงค่าเฉลี่ยของฝุ่น PM2.5 และ PM10 ทุกจุดในช่วงเวลาการศึกษา ได้แก่
เดือนมกราคม - พฤษภาคม 2567 รวมทั้งสิ้น 22 สัปดาห์

หน้า PREDICTION ออกแบบมาเพื่อนำโมเดลที่ถูกฝึกสอนมาใช้ในการทำนายประเภทของ
หุบเมืองในเมืองอื่น โดยการดึงข้อมูลภาพจาก Google Street View API และแสดงผลการทำนายเมื่อผู้ใช้คลิกจุดที่เลือก

หน้า ANOVA ออกแบบมาเพื่อแสดงผลการวิเคราะห์ความแปรปรวน (ANOVA) ของค่า
PM2.5 และ PM10 ในแต่ละประเภทของหุบเมือง โดยแสดงผลในรูปแบบกราฟ Boxplot

หน้า TUKEY'HSD ออกแบบมาเพื่อแสดงผลการวิเคราะห์ค่าความแตกต่างเฉลี่ยด้วยวิธี Tukey's HSD ของค่า PM2.5 และ PM10 ในแต่ละประเภทของหุบเมือง โดยแสดงผลใน
รูปแบบ Matrix Chart
